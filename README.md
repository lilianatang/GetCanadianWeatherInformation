# Get Canadian Weather Information
Python script to get Canadian weather information

Public Data Sources:
- https://climate.weather.gc.ca/climate_data/bulk_data_e.html that stores all the weather information by stations
- https://collaboration.cmc.ec.gc.ca/cmc/climate/Get_More_Data_Plus_de_donnees/Station%20Inventory%20EN.csv that stores all the station inventory.
